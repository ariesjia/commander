# 音效文件

## 盲盒开启音效 (box-open.mp3)

开盲盒时需要播放的音效，请将文件命名为 `box-open.mp3` 放在此目录。

### 推荐音效（免费可商用）

1. **Mixkit - Box 类音效**（推荐）
   - 打开：https://mixkit.co/free-sound-effects/box/
   - 推荐「Open special box」或「Greeting music box tone」
   - 点击 "Download Free SFX" 下载 MP3，重命名为 `box-open.mp3`

2. **Freesound - Treasure Chest Open**（宝箱开启，温馨）
   - 链接：https://freesound.org/people/steprock/sounds/771164/
   - 需注册后下载，CC0 协议
   - 重命名为 `box-open.mp3` 放入此目录
