## 二、外观设计

### 2.1 整体形态

- **主体结构**：轻量人形机甲，流线型，身高约 2.6～3 米，重心灵活
- **比例**：四肢修长，躯干紧凑，背部可展开风雷双翼
- **装甲**：轻量化，以机动与侦察为主

### 2.2 配色方案

- **主色**：浅灰（#a0a0a0～#c0c0c0）— 低调、融入环境
- **辅色**：天蓝（#4a90d9～#6eb5e8）— 翼装、传感器
- **点缀**：闪电白（#e0e0e0）— 推进器、能量纹路

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **头部** | 传感器阵列突出，多光谱侦察，视野广阔 |
| **躯干** | 紧凑装甲，背部集成翼装收纳舱 |
| **四肢** | 轻量化关节，足部带缓冲，利于复杂地形机动 |
| **翼装** | 风雷双翼可展开，支持短距滑翔与跨越地形 |

### 2.4 材质与质感

- 外壳：轻质复合装甲，表面光滑
- 翼装：柔性材料 + 推进器，可折叠收纳
- 传感器：多光谱镜头，可切换夜视/热成像

---

## 三、核心能力

### 3.1 高速移动

- 凭借轻量化与高机动关节，可高速穿梭战场
- 灵活穿梭复杂地形，获取情报
- 适合前沿探查、迂回牵制

### 3.2 战场侦察

- 多光谱传感器可获取敌方位置、阵型、火力配置
- 情报实时回传，为友军提供视野
- 是战场的「眼睛」

### 3.3 迂回牵制

- 可绕至敌阵侧翼，吸引火力或制造混乱
- 不正面交战，以机动与骚扰为主
- 为友军创造战机

### 3.4 快速支援

- 凭借高速机动，可快速抵达友军位置
- 提供情报、物资投送、或紧急支援
- 适合救援落单单位

## 四、专属装备

| 装备名 | 功能 |
|--------|------|
| **风雷双翼** | 背部翼装，支持短距滑翔与跨越地形 |
| **瞬影突袭** | 短时爆发推进，用于紧急撤离或快速切入 |
| **多光谱传感器** | 头部与躯干，夜视/热成像/多光谱切换 |
| **轻量化关节** | 四肢高响应关节，支持复杂地形机动 |

---

## 五、职责与战术

### 5.1 战场职责

- 前沿探查 — 率先抵达，获取情报
- 迂回牵制 — 绕至侧翼，吸引火力
- 快速支援 — 快速抵达，提供支援

### 5.2 战术风格

- **侦察优先**：以情报获取为主，不恋战
- **协同定位**：为利刃、俯冲等提供目标位置
- **机动游击**：不正面硬刚，以迂回与支援为主

### 5.3 适用地形

- 复杂地形、城市、山地、丛林
- 开阔战场（翼装可跨越障碍）
- 不适合：深海、地下（地渊负责）

## 六、等级进化体系

| 等级 | 称号 | 进化要点 |
|------|------|----------|
| 0 | 量产型 | 基础高速移动与侦察能力，可执行前沿探查任务 |
| 1 | 改良型 | 灵活穿梭复杂地形能力增强，情报获取效率提升 |
| 2 | 特装型 | 机动侦查能力初显，迂回牵制反应更快 |
| 3 | 强化型 | 战场侦察与快速支援能力成型 |
| 4 | 翼装型 | 风雷双翼加身，高速移动与跨越地形能力大幅增强 |
| 5 | 突袭型 | 瞬影突袭觉醒，迅捷之威尽显 |
| 6 | 疾风型 | 迅捷之力臻于完美，灵活穿梭与快速支援达到巅峰 |
| 7 | 完整体 | 机动侦查的终极形态，风雷双翼与瞬影突袭全开，成为战场最迅捷的侦察之眼 |

## 九、简介文案（用于系统）

**简短描述**：机动侦查型人形机甲，擅长高速移动与战场侦察。

**完整介绍**：风驰，机动侦查型人形机甲，擅长高速移动与战场侦察，灵活穿梭复杂地形获取情报。负责前沿探查、迂回牵制、快速支援。它在历练中不断觉醒，解锁更强形态、风雷双翼与瞬影突袭，每一次蜕变都让迅捷之力更强。

## 十、设计参考要点（供绘图/建模）

1. **迅捷感**：轻量化、流线型、翼装
2. **侦察感**：传感器突出、多角度视野、低调配色
3. **灵活感**：修长四肢、高响应关节、可折叠翼装
4. **差异化**：与利刃（突袭）、空境（载具）、俯冲（猛禽）在形态上明显区分
