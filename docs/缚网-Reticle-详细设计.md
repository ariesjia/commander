## 二、外观设计

### 2.1 整体形态

- **主体结构**：八足仿生蜘蛛形态，躯干呈扁圆流线型，便于穿越狭缝与管道
- **尺寸**：比人形机甲小一号，约 1.5～2 米体长，适合室内与废墟作战
- **关节**：多节肢结构，每足 3～4 段可折叠关节，支持 360° 攀附与倒挂

### 2.2 配色方案

- **主色**：暗紫（#2d1b4e～#4a2c6a）— 低调、隐蔽感
- **辅色**：金属银（#c0c0c0～#e8e8e8）— 关节、足尖、传感器
- **点缀**：深红（#8b0000）— 复眼、丝线发射口，用于警示与识别

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **头部** | 复眼阵列（多光谱传感器），口器位置为纳米丝发射器 |
| **躯干** | 流线型装甲，背部可展开「丝腺」模块，储存与合成纳米丝 |
| **八足** | 足尖带吸附垫与微钩，可攀附玻璃、金属、混凝土；关节处有静音缓冲 |
| **腹部** | 丝线储存舱，可快速释放预制丝网 |

### 2.4 材质与质感

- 外壳：哑光复合装甲，减少反光与雷达反射
- 足部：柔性吸附材质 + 金属骨架
- 丝线：半透明纳米纤维，肉眼难辨，可导电/传感

## 三、核心能力

### 3.1 垂直攀爬

- 可在墙面、天花板、管道内壁自由移动
- 吸附力可调：静默模式（轻附）与强力模式（负重/战斗）
- 移动时几乎无声，适合潜入

### 3.2 丝网布设

- **侦测丝网**：细丝布于通道，触碰到即回传位置与震动信息
- **束缚丝网**：高强度丝线可缠住目标，限制移动
- **陷阱丝网**：触发式，可连接警报或小型爆炸装置

### 3.3 狭缝渗透

- 躯体可压缩变形，穿越 30cm 级缝隙
- 八足可收拢为「梭形」，在管道中快速滑行

### 3.4 隐蔽侦察

- 复眼支持夜视、热成像、多光谱
- 可倒挂于天花板，长时间静默观测

## 四、专属装备

| 装备名 | 功能 |
|--------|------|
| **纳米丝发射器** | 口器位置，可发射单丝、丝网、丝弹；丝线可导电，用于干扰或标记 |
| **多关节攀附足** | 八足末端，吸附垫 + 微钩 + 静音关节，适应多种表面 |
| **丝腺模块** | 背部可展开，合成纳米丝并储存；支持快速补丝 |
| **复眼传感器阵列** | 360° 视野，夜视/热成像/多光谱切换 |

## 五、职责与战术

### 5.1 战场职责

- 垂直攀爬 — 占领制高点、天花板伏击点
- 狭缝渗透 — 穿越废墟、管道、通风系统
- 陷阱布设 — 在关键通道设侦测网与束缚网
- 隐蔽侦察 — 敌后静默观测，回传情报

### 5.2 战术风格

- **渗透优先**：避免正面交战，以侦察与控场为主
- **协同定位**：为利刃、猎犬等突击单位标记路线与陷阱
- **区域封锁**：在撤离路线布设丝网，延缓追兵

### 5.3 适用地形

- 城市废墟、室内、管道、通风系统
- 山地洞穴、隧道入口
- 不适合：开阔平原、深海（地渊负责）、高空（空境负责）

## 六、等级进化体系

| 等级 | 称号 | 进化要点 |
|------|------|----------|
| 0 | 量产型 | 基础八足攀爬与单丝发射，可执行简单渗透 |
| 1 | 改良型 | 丝网布设能力增强，吸附足静音优化 |
| 2 | 特装型 | 复眼多光谱解锁，侦测丝网初显 |
| 3 | 强化型 | 躯体压缩变形，可穿越更窄缝隙 |
| 4 | 织网型 | 丝腺模块加身，丝网范围与强度大幅提升 |
| 5 | 潜蛛型 | 束缚丝网与陷阱丝网成型，控场之威尽显 |
| 6 | 蛛后型 | 缚网之力臻于完美，狭小空间渗透达到巅峰 |
| 7 | 完整体 | 蛛型渗透的终极形态，纳米丝发射器与缚网之力全开，成为狭小空间渗透的专家 |

## 七、人物特质（可选）

- **性格**：冷静、耐心、善于等待时机
- **战斗风格**：不喜正面冲突，偏好布网后静待猎物
- **与驾驶员**：需要高度默契，渗透任务往往单独执行，依赖信任

## 九、简介文案（用于系统）

**简短描述**：仿生蛛型渗透机甲，擅长垂直攀爬与丝网布设。

**完整介绍**：缚网是狭小空间渗透单位，凭借多足攀爬与丝网布设，在建筑与废墟中执行渗透与侦察任务。负责垂直攀爬、狭缝渗透、陷阱布设、隐蔽侦察。它在历练中不断觉醒，解锁更强形态、纳米丝发射器与缚网之力，每一次蜕变都让渗透之力更强。

## 十、设计参考要点（供绘图/建模）

1. **蜘蛛感**：八足、复眼、丝线，但保持机械感，避免过于生物化
2. **隐蔽感**：暗色、哑光、低轮廓
3. **精密感**：足尖吸附垫、关节结构、丝腺模块的机械细节
4. **差异化**：与地渊（蚯蚓）、猎犬（犬）、俯冲（鸟）在形态上明显区分
