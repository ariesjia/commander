## 种子数据片段（示例）

`imageUrl` 使用站点根路径，与现有 `MechaLevel.imageUrl` 风格一致：

```json
[
  {
    "slug": "pulse-ion-cannon",
    "name": "脉冲离子炮",
    "description": "重型机甲用高输出能量炮。炮口六角约束环内蓄积高密度离子脉冲，可贯穿复合装甲；能耗较高，适合一击决胜的火力窗口。",
    "imageUrl": "/item/0001.png",
    "kind": "DISPLAY"
  }
]
```

其余条目将 `slug`、`name`、`description` 与上表对应，`imageUrl` 为 `/item/0003.png` … `/item/0016.png`（按文件名）。
