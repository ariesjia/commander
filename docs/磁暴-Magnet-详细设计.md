# 磁暴 (Magnet) — 详细设计文档

> 磁场控制型人形机甲 · 引力之核

---

## 一、基础设定

| 项目 | 内容 |
|------|------|
| **中文名** | 磁暴 |
| **英文名** | Magnet |
| **类型** | 磁场控制型人形机甲 |
| **定位** | 装备干扰与阵型打乱单位 |
| **设计灵感** | 电磁、引力（控制、干扰、束缚） |

---

## 二、外观设计

### 2.1 整体形态

- **主体结构**：标准人形机甲，躯干略宽，身高约 3～3.5 米，背部有磁场发生器
- **比例**：双臂可展开磁场发射装置，整体呈「磁极」姿态
- **装甲**：抗磁复合装甲，可承受自身磁场反噬

### 2.2 配色方案

- **主色**：深灰（#4a4a4a～#6b6b6b）— 金属、磁场感
- **辅色**：电蓝（#3182ce～#63b3ed）— 磁场能量、纹路
- **点缀**：磁极红蓝（#e53e3e / #3182ce）— 双极标识、核心

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **头部** | 护面带磁场纹路，传感器可探测金属与能量 |
| **躯干** | 胸甲有引力核心，可释放磁场波 |
| **双臂** | 磁场发射装置，可定向释放引力/斥力 |
| **背部** | 磁场发生器，扩大磁场范围与强度 |

### 2.4 材质与质感

- 外壳：抗磁复合装甲，表面有磁场纹路
- 核心：双极引力核心，红蓝双色辉光
- 发射装置：金属质感，释放时呈蓝白磁场波纹

---

## 三、核心能力

### 3.1 引力牵引

- 可对金属目标施加引力，牵引或拉近
- 可改变敌方机甲移动轨迹，打乱阵型
- 适合配合利刃、猎犬实施集火

### 3.2 能量干扰

- 磁场可干扰敌方电子设备、传感器、通讯
- 装备干扰可使敌机甲暂时失灵
- 适合削弱敌方火力与侦察能力

### 3.3 引力束缚

- 可对目标施加持续引力，限制移动
- 束缚轻型单位，为友军创造输出机会
- 适合控场与定点清除

### 3.4 阵型打乱

- 引力/斥力可打乱敌方阵型
- 使敌人无法保持有效编队
- 配合玄甲、雷啸形成控场+输出

## 四、专属装备

| 装备名 | 功能 |
|--------|------|
| **引力核心** | 胸甲内双极核心，可释放引力/斥力场 |
| **磁暴装甲** | 抗磁装甲，可承受自身磁场反噬 |
| **磁场发射装置** | 双臂，定向释放磁场波 |
| **双极传感器** | 可探测金属与能量目标，锁定牵引 |

---

## 五、职责与战术

### 5.1 战场职责

- 装备干扰 — 干扰敌方电子设备与通讯
- 引力束缚 — 限制目标移动
- 阵型打乱 — 打乱敌方编队

### 5.2 战术风格

- **控场优先**：以干扰与束缚为主，不直接输出
- **协同定位**：为利刃、猎犬、雷啸创造输出机会
- **辅助核心**：改变战场态势，限制敌人行动

### 5.3 适用地形

- 开阔战场、阵地战、金属密集区域
- 不适合：非金属密集区域（效果减弱）、强电磁干扰环境

## 六、等级进化体系

| 等级 | 称号 | 进化要点 |
|------|------|----------|
| 0 | 量产型 | 基础引力牵引与能量干扰能力，可执行装备干扰任务 |
| 1 | 改良型 | 磁场控制能力增强，双极之力初绽 |
| 2 | 特装型 | 引力束缚能力初显，可限制敌人行动 |
| 3 | 强化型 | 改变战场态势能力成型，干扰与控场之力涌动 |
| 4 | 磁能型 | 磁能武装加身，引力牵引与能量干扰范围大幅增强 |
| 5 | 引力型 | 引力核心觉醒，阵型打乱之威尽显 |
| 6 | 核心型 | 磁力之力臻于完美，磁场控制达到巅峰 |
| 7 | 完整体 | 磁场控制的终极形态，引力核心与磁能武装全开，成为战场控场的绝对主宰 |

## 七、人物特质（可选）

- **性格**：冷静、理性、善于分析战场态势
- **战斗风格**：不争输出，以控场与干扰为主
- **与驾驶员**：需要默契，磁场释放时机需配合友军

## 九、简介文案（用于系统）

**简短描述**：磁场控制型人形机甲，擅长引力牵引与能量干扰。

**完整介绍**：磁暴，磁场控制型人形机甲，擅长引力牵引与能量干扰，可改变战场态势限制敌人行动。负责装备干扰、引力束缚、阵型打乱。它在历练中不断觉醒，解锁更强形态、磁暴装甲与引力核心，每一次蜕变都让磁力之力更强。

## 十、设计参考要点（供绘图/建模）

1. **磁场感**：双极纹路、磁场波纹、红蓝核心
2. **控制感**：发射装置、引力核心、宽肩
3. **科技感**：抗磁装甲、传感器、能量纹路
4. **差异化**：与雷啸（雷电）、星盾（屏障）在形态上明显区分
