# 雷啸 (Thunder) — 详细设计文档

> 范围爆发型人形机甲 · 雷霆之怒

---

## 一、基础设定

| 项目 | 内容 |
|------|------|
| **中文名** | 雷啸 |
| **英文名** | Thunder |
| **类型** | 范围爆发型人形机甲 |
| **定位** | 群体伤害与区域控场单位 |
| **设计灵感** | 雷神、风暴（雷电、范围、压制） |

---

## 二、外观设计

### 2.1 整体形态

- **主体结构**：标准人形机甲，肩部宽厚，身高约 3～3.5 米，背部有能量传导装置
- **比例**：上肢略粗，可承载雷电武装，整体呈「风暴核心」姿态
- **装甲**：绝缘复合装甲，可承受自身雷电反噬

### 2.2 配色方案

- **主色**：深蓝（#1e3a5f～#2c5282）— 风暴、雷霆感
- **辅色**：银白（#e8e8e8～#f5f5f5）— 雷电能量、纹路
- **点缀**：雷霆金（#f0c14b）— 电弧、核心、武装

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **头部** | 护面带能量纹路，眼部呈电弧辉光 |
| **躯干** | 胸甲有雷电核心，可释放大范围雷电 |
| **双臂** | 雷电武装，可发射雷球、雷链、雷暴 |
| **背部** | 能量传导装置，连接雷电核心与武装 |

### 2.4 材质与质感

- 外壳：绝缘复合装甲，表面有能量纹路
- 武装：雷电发生器，电弧呈金白色
- 核心：雷电能量源，释放时全身纹路发光

---

## 三、核心能力

### 3.1 大范围雷电攻击

- 可释放雷球、雷链、雷暴，覆盖整片战场
- 群体伤害，适合压制密集敌阵
- 雷电可传导，对金属目标伤害加成

### 3.2 场面压制

- 持续雷电输出可压制敌方火力与机动
- 区域控场，为友军创造输出空间
- 适合阵地战、防守战

### 3.3 区域控场

- 可在指定区域布设雷电场，限制敌人移动
- 雷电可麻痹轻型单位，延缓敌方行动
- 配合星盾、玄甲形成攻防一体

### 3.4 群体伤害

- 以强力输出覆盖整片战场
- 适合清理杂兵、压制火力点
- 是团队主要的范围输出单位

## 四、专属装备

| 装备名 | 功能 |
|--------|------|
| **雷电武装** | 双臂雷电发生器，可发射雷球、雷链、雷暴 |
| **爆轰之力** | 核心强化，大范围雷电伤害与控场能力提升 |
| **雷电核心** | 胸甲内能量源，支撑长时间雷电输出 |
| **绝缘装甲** | 全身绝缘，可承受自身雷电反噬 |

---

## 五、职责与战术

### 5.1 战场职责

- 群体伤害 — 大范围雷电输出
- 场面压制 — 压制敌方火力与机动
- 区域控场 — 布设雷电场，限制敌人

### 5.2 战术风格

- **范围优先**：以覆盖整片战场为主
- **协同定位**：需星盾/玄甲掩护，雷啸负责输出
- **控场辅助**：雷电麻痹可配合利刃、猎犬突袭

### 5.3 适用地形

- 开阔战场、阵地战、防守战
- 不适合：狭窄空间（易误伤友军）、水下、导电环境

## 六、等级进化体系

| 等级 | 称号 | 进化要点 |
|------|------|----------|
| 0 | 量产型 | 基础雷电攻击能力，可执行群体伤害任务 |
| 1 | 改良型 | 大范围雷电攻击能力增强，雷能初绽 |
| 2 | 特装型 | 场面压制能力初显，可覆盖整片战场 |
| 3 | 强化型 | 强力输出与区域控场能力成型，雷霆之威尽显 |
| 4 | 轰雷型 | 雷电武装加身，群体伤害与控场范围大幅增强 |
| 5 | 万钧型 | 爆轰之力全开，范围爆发之威尽显 |
| 6 | 雷翼型 | 轰击之力臻于完美，大范围雷电攻击与群体控制达到巅峰 |
| 7 | 完整体 | 范围爆发的终极形态，雷电武装与爆轰之力全开，成为战场范围压制的雷霆核心 |

## 七、人物特质（可选）

- **性格**：张扬、果断、追求爆发
- **战斗风格**：以范围压制为主，不惧正面交锋
- **与驾驶员**：需要默契，释放时机需配合友军位置

## 九、简介文案（用于系统）

**简短描述**：范围爆发型人形机甲，擅长大范围雷电攻击与群体控制。

**完整介绍**：雷啸，范围爆发型人形机甲，擅长大范围雷电攻击与群体控制，以强力输出覆盖整片战场。负责群体伤害、场面压制、区域控场。它在历练中不断觉醒，解锁更强形态、雷电武装与爆轰之力，每一次蜕变都让轰击之力更强。

## 十、设计参考要点（供绘图/建模）

1. **雷霆感**：电弧纹路、雷电武装、能量核心
2. **爆发感**：宽肩、能量传导装置、金白辉光
3. **风暴感**：深蓝配色、雷电纹路、护面电弧
4. **差异化**：与磁暴（磁场）、利刃（单体）在形态上明显区分
