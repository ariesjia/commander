# 利刃 (Razor) — 详细设计文档

> 极速突袭型人形机甲 · 斩击之锋

---

## 一、基础设定

| 项目 | 内容 |
|------|------|
| **中文名** | 利刃 |
| **英文名** | Razor |
| **类型** | 极速突袭型人形机甲 |
| **定位** | 快速突袭与弱点击破单位 |
| **设计灵感** | 刺客、忍者（极速、精准、一击必杀） |

---

## 二、外观设计

### 2.1 整体形态

- **主体结构**：纤细人形机甲，流线型，身高约 2.8～3.2 米，重心灵活
- **比例**：四肢修长，躯干紧凑，整体呈「刀刃」般锐利线条
- **装甲**：轻量化，以机动为主，减少负重

### 2.2 配色方案

- **主色**：银灰（#808080～#a0a0a0）— 低调、迅捷感
- **辅色**：深蓝（#2c5282～#3182ce）— 能量刃、关节
- **点缀**：雷霆金（#f0c14b）— 刃锋、核心

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **头部** | 狭长流线型，传感器多角度布置，视野广阔 |
| **躯干** | 紧凑装甲，核心舱居中，可快速散热 |
| **四肢** | 关节灵活，足部轻量化，利于高速移动与急停 |
| **武装** | 双臂可装备破甲刃/能量刃，可折叠收纳 |

### 2.4 材质与质感

- 外壳：轻质复合装甲，表面光滑减阻
- 刃锋：高密度合金 + 能量附着，切割力极强
- 关节：高响应液压，支持瞬间爆发

---

## 三、核心能力

### 3.1 瞬间切入

- 凭借极速机动，可快速切入敌阵薄弱环节
- 移动轨迹难以预判，适合绕后与侧翼突袭
- 切入后立即实施打击，不恋战

### 3.2 精准打击

- 锁定目标弱点（关节、能源舱、传感器）
- 以超高效率瓦解敌方关键目标
- 一击即退，或连续斩击后撤离

### 3.3 弱点击破

- 破甲武装可穿透轻型装甲
- 擅长斩断敌机甲关节、破坏关键系统
- 为友军创造突破口

### 3.4 敌后干扰

- 可深入敌后，袭击补给、通讯、指挥节点
- 制造混乱，牵制敌方兵力
- 配合正面部队形成夹击

---

## 四、专属装备

| 装备名 | 功能 |
|--------|------|
| **破甲武装** | 双臂能量刃，可穿透轻型装甲，切割关节 |
| **雷霆锋芒** | 刃锋能量强化，斩击时附带电弧，增加伤害 |
| **高机动关节** | 四肢关节，支持极速移动与急停转向 |
| **多角度传感器** | 头部与躯干，全方位视野，利于切入与撤离 |

---

## 五、职责与战术

### 5.1 战场职责

- 快速突袭 — 切入敌阵薄弱环节
- 弱点击破 — 瓦解关键目标
- 敌后干扰 — 袭击补给与指挥节点

### 5.2 战术风格

- **一击即退**：切入→打击→撤离，不恋战
- **协同定位**：为玄甲、雷啸等创造突破口
- **游击战术**：以骚扰与斩首为主

### 5.3 适用地形

- 城市、废墟、复杂地形
- 开阔战场（需星盾/玄甲吸引火力）
- 不适合：正面硬刚（玄甲更优）、深海、高空

## 六、等级进化体系

| 等级 | 称号 | 进化要点 |
|------|------|----------|
| 0 | 量产型 | 基础极速与精准打击能力，可执行快速突袭任务 |
| 1 | 改良型 | 瞬间切入能力增强，弱点击破效率提升 |
| 2 | 特装型 | 斩击之力初显，以超高效率瓦解关键目标 |
| 3 | 强化型 | 极速突袭能力成型，敌后干扰反应更快 |
| 4 | 破甲型 | 破甲武装加身，精准打击穿透力大幅增强 |
| 5 | 雷霆型 | 雷霆锋芒觉醒，斩击之威尽显 |
| 6 | 锋芒型 | 斩击之力臻于完美，瞬间切入与精准打击达到巅峰 |
| 7 | 完整体 | 极速突袭的终极形态，破甲武装与雷霆锋芒全开，成为战场斩首的绝对利刃 |

## 七、人物特质（可选）

- **性格**：冷静、果断、追求效率
- **战斗风格**：迅捷如风，一击必中
- **与驾驶员**：需要高度默契，切入与撤离时机需精准配合

## 九、简介文案（用于系统）

**简短描述**：极速突袭型人形机甲，擅长瞬间切入与精准打击。

**完整介绍**：利刃，极速突袭型人形机甲，擅长瞬间切入与精准打击，以超高效率瓦解敌方关键目标。负责快速突袭、弱点击破、敌后干扰。它在历练中不断觉醒，解锁更强形态、破甲武装与雷霆锋芒，每一次蜕变都让斩击之力更强。

## 十、设计参考要点（供绘图/建模）

1. **锐利感**：流线型、刀刃线条、纤细四肢
2. **迅捷感**：轻量化、关节灵活、能量刃
3. **刺客感**：低调配色、多角度传感器、可折叠武装
4. **差异化**：与玄甲（重装）、星盾（防御）、猎犬（四足）在形态上明显区分
