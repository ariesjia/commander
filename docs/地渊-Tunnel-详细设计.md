# 地渊 (Tunnel) — 详细设计文档

> 地下隧道仿生机械兽 · 地下专家

---

| 项目 | 内容 |
|------|------|
| **中文名** | 地渊 |
| **英文名** | Tunnel |

| **定位** | 地下渗透与隧道开辟单位 |

## 二、外观设计

### 2.1 整体形态

- **主体结构**：仿蚯蚓/穿山甲形态，长筒形分段躯体，体长约 4～6 米，直径约 0.8～1.2 米
- **比例**：头部尖锐，躯干分节，每节可独立伸缩与旋转
- **关节**：多节环形结构，支持波浪式推进与钻头旋转

### 2.2 配色方案

- **主色**：深褐（#3d2914～#5c4033）— 泥土、岩层融入感
- **辅色**：暗金（#8b7355～#a08060）— 金属质感、钻头、关节
- **点缀**：琥珀橙（#d4a017）— 传感器、动力舱

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **头部** | 尖锐钻头形态，地脉钻头可旋转破岩，口部为岩屑排出通道 |
| **躯干** | 分节环形结构，每节有独立推进器与吸附垫，支持波浪式前进 |
| **体表** | 鳞片状装甲，可反向竖起增加与岩壁的摩擦力 |
| **尾部** | 可展开稳定鳍，在隧道中提供推进与方向控制 |

### 2.4 材质与质感

- 外壳：耐磨复合装甲，适应岩层摩擦
- 钻头：超硬合金，可更换不同岩层适配钻头
- 体表：鳞片可调角度，兼具防护与推进

---

## 三、核心能力

### 3.1 钻地潜行

- 地脉钻头旋转破岩，躯体波浪式推进
- 可在土壤、岩层、冻土中穿行
- 潜行时几乎无地表震动，隐蔽性强

### 3.2 隧道开辟

- 钻出的通道可供友军通过
- 可预设隧道走向，连接地下设施或绕过地表防线
- 隧道内壁可喷涂加固剂，提高稳定性

### 3.3 隐蔽运输

- 隧道内可运输小型单位或物资
- 地渊自身也可携带少量关键物资
- 适合渗透、撤离、补给线建立

### 3.4 地下侦察

- 声波探地仪可探测前方岩层结构、空洞、敌方地下设施
- 可布设地下传感器节点，建立地下情报网
- 为友军提供地下战场态势

## 四、专属装备

| 装备名 | 功能 |
|--------|------|
| **地脉钻头** | 头部旋转钻头，可更换岩层适配钻头，破岩效率高 |
| **声波探地仪** | 头部传感器，探测前方岩层结构、空洞、地下设施 |
| **分节推进器** | 躯干每节独立推进，波浪式前进，适应不同地质 |
| **鳞片装甲** | 体表可调鳞片，增加摩擦力与防护 |

## 五、职责与战术

### 5.1 战场职责

- 地下渗透 — 从地下潜入敌后或目标区域
- 隧道开辟 — 为友军开辟隐蔽通道
- 隐蔽运输 — 在隧道内运输人员与物资
- 地下侦察 — 探测地下结构，建立情报网

### 5.2 战术风格

- **地道战**：避开地表防线，从地下直达目标
- **协同定位**：为缚网、利刃等提供地下通道与情报
- **持久作战**：隧道可重复使用，支持多次渗透与撤离

### 5.3 适用地形

- 土壤、岩层、山地、丘陵
- 城市地下、废弃矿坑、洞穴系统
- 不适合：深海（沧龙负责）、流沙、极硬岩层（需特殊钻头）

## 六、等级进化体系

| 等级 | 称号 | 进化要点 |
|------|------|----------|
| 0 | 量产型 | 基础钻地潜行能力，可执行地下渗透任务 |
| 1 | 改良型 | 隧道开辟效率提升，岩屑排出优化 |
| 2 | 特装型 | 地脉钻头初显，可应对更硬岩层 |
| 3 | 强化型 | 声波探地仪解锁，地下侦察能力成型 |
| 4 | 深潜型 | 地脉钻头加身，隧道开辟与隐蔽通道能力大幅增强 |
| 5 | 地脉型 | 分节推进器强化，地下机动之威尽显 |
| 6 | 裂隙型 | 地渊之力臻于完美，为队伍开辟隐蔽通道达到巅峰 |
| 7 | 完整体 | 地下渗透的终极形态，地脉钻头与深潜之力全开，成为地下战场开辟通道的专家 |

## 七、人物特质（可选）

- **性格**：沉稳、坚韧、耐得住地下黑暗与孤寂
- **战斗风格**：不争地表锋芒，默默于地下开辟道路
- **与驾驶员**：需要高度信任，地下任务往往长时间单独执行

---

## 八、与其他机甲的协同

| 机甲 | 协同方式 |
|------|----------|
| 缚网 | 地渊开辟地下通道，缚网负责通道内侦察与陷阱布设 |
| 利刃 | 地渊打通敌后隧道，利刃从隧道突袭 |
| 方舟 | 地渊开辟补给线隧道，方舟通过隧道投送物资 |
| 沧龙 | 地渊负责陆地地下，沧龙负责水域，二者可衔接两栖渗透 |

## 九、简介文案（用于系统）

**简短描述**：地下隧道仿生机械兽，擅长钻地潜行与隧道开辟。

**完整介绍**：地渊是地下战场的专家，仿蚯蚓/穿山甲形态，可钻地潜行、打通隧道，在复杂地下环境中快速机动，为队伍开辟路线。负责地下渗透、隧道开辟、隐蔽运输、地下侦察。它在历练中不断觉醒，解锁更强形态、地脉钻头与深潜之力，每一次蜕变都让地下之力更强。

## 十、设计参考要点（供绘图/建模）

1. **蚯蚓/穿山甲感**：长筒分节、头部钻头、鳞片体表
2. **地下感**：深褐配色、耐磨质感、岩屑痕迹
3. **机械感**：钻头金属光泽、分节关节、推进器喷口
4. **差异化**：与猎犬（四足）、俯冲（鸟）、缚网（蜘蛛）在形态上明显区分
