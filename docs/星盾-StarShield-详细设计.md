# 星盾 (Star Shield) — 详细设计文档

> 全能防御型人形机甲 · 守护之壁

---

## 二、外观设计

### 2.1 整体形态

- **主体结构**：标准人形机甲，体态稳健，身高约 3～4 米，重心略后倾
- **比例**：肩宽与躯干厚实，持盾侧略前倾，整体呈「守护」姿态
- **装甲**：全身均衡防护，持盾臂与胸甲尤为厚重

### 2.2 配色方案

- **主色**：银白（#e8e8e8～#f5f5f5）— 圣洁、守护感
- **辅色**：天蓝（#4a90d9～#6eb5e8）— 屏障能量、纹饰
- **点缀**：星辰金（#d4af37）— 盾面、核心、纹章

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **头部** | 护面温和，眼部传感器呈蓝光，传达守护意志 |
| **躯干** | 胸甲宽阔，核心舱可展开光之铠甲能量 |
| **持盾臂** | 超大型臂盾，可展开为区域屏障 |
| **足部** | 稳固底盘，可展开锚定装置，驻守时难以撼动 |

### 2.4 材质与质感

- 外壳：复合装甲 + 能量护盾发生器
- 盾面：可展开光之屏障，呈半透明蓝白色
- 纹饰：星辰纹章，象征守护与希望

---

## 三、核心能力

### 3.1 绝对屏障

- 可展开单体或小范围能量屏障，抵御高强度攻击
- 屏障可持续维持，为友军创造安全输出环境
- 适合抵挡集火、保护关键单位

### 3.2 群体庇护

- 屏障范围可扩展，覆盖多名友军
- 区域防御能力可保护阵地或撤离路线
- 是团队最可靠的「移动掩体」

### 3.3 承伤掩护

- 可主动吸引火力，承受攻击
- 凭借厚重装甲与屏障，稳定承受高强度攻击
- 为友军争取输出与撤离时间

### 3.4 危机救援

- 可快速移动至友军身旁展开屏障
- 救援反应快，适合保护落单或受伤单位

## 五、职责与战术

### 5.1 战场职责

- 承伤掩护 — 吸引火力，承受攻击
- 区域防御 — 构筑屏障，保护友军
- 危机救援 — 快速抵达，展开庇护

### 5.2 战术风格

- **守护优先**：以友军安全为第一要务
- **协同定位**：为玄甲、利刃、雷啸等创造输出空间
- **稳守反击**：不主动进攻，以防御与掩护为主

### 5.3 适用地形

- 阵地战、开阔战场、撤离路线
- 不适合：需要高机动的突袭任务（风驰/利刃更优）

## 六、等级进化体系

| 等级 | 称号 | 进化要点 |
|------|------|----------|
| 0 | 量产型 | 基础屏障与庇护能力，可承担承伤掩护任务 |
| 1 | 改良型 | 守护之力凝实，能为团队创造更安全的输出环境 |
| 2 | 特装型 | 区域防御能力初显，可稳定承受一定强度攻击 |
| 3 | 强化型 | 绝对屏障构筑能力增强，危机救援反应更快 |
| 4 | 光铠型 | 光之铠甲加身，群体庇护范围与强度大幅提升 |
| 5 | 星辰型 | 星辰力量觉醒，全能防御之威尽显 |
| 6 | 星辉型 | 守护之力臻于完美，构筑绝对屏障与群体庇护 |
| 7 | 完整体 | 全能防御的终极形态，光之铠甲与星辰屏障全开，成为团队最可靠的守护之壁 |

## 七、人物特质（可选）

- **性格**：沉稳、可靠、以守护为使命

- **与驾驶员**：需要高度信任，守护时机需精准判断

## 九、简介文案（用于系统）

**简短描述**：全能防御型人形机甲，擅长构筑绝对屏障与群体庇护。

**完整介绍**：星盾，全能防御型人形机甲，擅长构筑绝对屏障与群体庇护，能稳定承受高强度攻击，为团队创造安全输出环境。负责承伤掩护、区域防御、危机救援。它在历练中不断觉醒，解锁更强形态、光之铠甲与星辰力量，每一次蜕变都让守护之力更强。

## 十、设计参考要点（供绘图/建模）

1. **守护感**：盾形、宽阔胸甲、稳健姿态
2. **圣洁感**：银白配色、星辰纹章、蓝光屏障
3. **可靠感**：厚重装甲、锚定足、稳固底盘
4. **差异化**：与玄甲（进攻）、镇岳（载具堡垒）在形态上明显区分
