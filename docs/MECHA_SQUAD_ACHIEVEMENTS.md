# 机甲小队成就设计

基于 `docs/` 中各机甲详细设计的**人物特质**与**与其他机甲的协同**，设计机甲小队成就。学生收集指定组合的**全部**机甲（领养）后解锁对应成就。仅包含真正的机甲协同组合，不含数量类成就。

---

## 二、数据结构建议

```ts
// prisma/seed-data/mecha-squads.ts 或类似

export interface MechaSquadConfig {
  slug: string;
  name: string;
  description: string;
  comboSkill: string;        // 联合技名称
  comboSkillDescription: string;  // 联合技描述
  mechaSlugs: string[];     // 需全部收集才解锁
  sortOrder: number;
}

export const MECHA_SQUAD_ACHIEVEMENTS: MechaSquadConfig[] = [
  { slug: "assault-team", name: "突击小队", description: "...", comboSkill: "破阵连环", comboSkillDescription: "...", mechaSlugs: ["xuanjia", "razor", "hound"], sortOrder: 0 },
  // ... 共 12 个组合成就
];
```

## 三、解锁判定逻辑

`studentMechas` 中的 `mechaSlug` 包含该成就 `mechaSlugs` 的**全部**即解锁。

## 五、说明

- 成就 1～4 仅使用**已入 seed** 的机甲（xuanjia, star-shield, razor, swift, thunder, magnet, aether, tidal, titan-fort, tunnel, ark, medecac）
- 成就 5～12 包含**设计阶段**机甲（hound, dive, reticle, hive, net-weaver, sand-stalker），需先将其加入 seed 后成就方可生效
- 可根据实际入 seed 的机甲调整成就组合
