# 猎犬 (Hound) — 详细设计文档

> 四足突击型仿生机甲 · 战场猎手

---

## 一、基础设定

| 项目 | 内容 |
|------|------|
| **中文名** | 猎犬 |
| **英文名** | Hound |
| **类型** | 四足突击型仿生机甲 |
| **定位** | 快速突击与单点歼灭单位 |
| **设计灵感** | 军犬、猎犬（冲刺、撕咬、追踪） |

---

## 二、外观设计

### 2.1 整体形态

- **主体结构**：四足仿犬科形态，躯干呈流线型，肩高约 1.2～1.5 米，体长约 2 米
- **比例**：前肢略粗于后肢，肩部隆起，背部线条前倾，利于冲刺
- **关节**：四足各 3 段关节，支持高速奔跑、急停、转向与扑击

### 2.2 配色方案

- **主色**：哑光黑（#1a1a1a～#2d2d2d）— 低调、夜战友好
- **辅色**：暗红（#8b0000～#a52a2a）— 眼部、利齿、关节动力舱
- **点缀**：金属银（#808080）— 足尖、利齿刃面

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **头部** | 狭长流线型，口部为等离子利齿模块，眼部为夜视/热成像传感器 |
| **躯干** | 紧凑装甲，背部集成冲刺推进器，腹部有散热格栅 |
| **四足** | 足尖带防滑纹与爪钩，可抓地急停、攀爬低矮障碍 |
| **尾部** | 短尾平衡杆，末端可展开辅助推进器 |

### 2.4 材质与质感

- 外壳：哑光复合装甲，减少反光
- 关节：高强度合金 + 液压缓冲
- 利齿：等离子能量刃，刃面呈暗红辉光

---

## 三、核心能力

### 3.1 高速冲刺

- 四足爆发式推进，短距离内可达极高速度
- 背部冲刺推进器可短时爆发，用于扑击或紧急撤离
- 适合城市街道、废墟、复杂地形的快速机动

### 3.2 近战撕咬

- 等离子利齿可咬穿轻型装甲，对目标造成持续伤害
- 扑击动作：冲刺→跃起→咬合，一气呵成
- 擅长压制落单目标、瘫痪敌机甲关节

### 3.3 敌后骚扰

- 凭借机动性绕至敌阵侧翼或后方
- 咬断补给线、干扰通讯节点、袭击落单单位
- 一击即退，不恋战

### 3.4 单点歼灭

- 锁定薄弱目标（如侦察单位、后勤单位）后快速突进
- 以近战撕咬为主，配合冲刺推进器完成击杀
- 适合斩首、破袭任务

## 四、专属装备

| 装备名 | 功能 |
|--------|------|
| **等离子利齿** | 口部模块，咬合时释放等离子能量，可切割装甲、瘫痪关节 |
| **冲刺推进器** | 背部与尾端，短时爆发推进，用于扑击加速或紧急撤离 |
| **足尖爪钩** | 四足末端，抓地、攀爬、急停时提供稳定支撑 |
| **夜视/热成像传感器** | 头部眼部，支持夜间与恶劣环境作战 |

---

## 五、职责与战术

### 5.1 战场职责

- 快速突击 — 率先切入敌阵薄弱环节
- 近战压制 — 以撕咬限制目标行动
- 敌后骚扰 — 绕后打击补给与落单单位
- 单点歼灭 — 锁定关键目标快速歼灭

### 5.2 战术风格

- **猎手型**：锁定目标→冲刺→扑击→咬合，一击即中
- **协同定位**：为利刃、玄甲等主攻单位创造突破口
- **游击战术**：不正面硬刚，以骚扰与突袭为主

### 5.3 适用地形

- 城市废墟、街道、室内
- 山地、丛林、复杂地形
- 不适合：开阔平原（易被集火）、深海、高空

## 六、等级进化体系

| 等级 | 称号 | 进化要点 |
|------|------|----------|
| 0 | 量产型 | 基础四足冲刺与利齿撕咬，可执行突击任务 |
| 1 | 改良型 | 冲刺推进器初显，机动性增强 |
| 2 | 特装型 | 等离子利齿强化，咬合穿透力提升 |
| 3 | 强化型 | 扑击动作成型，单点歼灭效率显著提升 |
| 4 | 追猎型 | 冲刺推进器加身，追击与撤离能力大幅增强 |
| 5 | 狂犬型 | 等离子利齿全开，近战压制之威尽显 |
| 6 | 猎王型 | 猎犬之力臻于完美，快速突击与单点歼灭达到巅峰 |
| 7 | 完整体 | 四足突击的终极形态，等离子利齿与冲刺推进器全开，成为战场追击与歼灭的猎手 |

## 七、人物特质（可选）

- **性格**：专注、果断、锁定目标后绝不松口
- **战斗风格**：迅猛、直接，偏好近战撕咬
- **与驾驶员**：需要高度默契，突击时机往往转瞬即逝

## 九、简介文案（用于系统）

**简短描述**：四足突击型仿生机甲，擅长高速冲刺与近战撕咬。

**完整介绍**：猎犬是战场上的四足猎手，凭借高速机动与近战能力，专攻落单与薄弱目标，是突击与追击的利器。负责快速突击、近战压制、敌后骚扰、单点歼灭。它在历练中不断觉醒，解锁更强形态、等离子利齿与冲刺推进器，每一次蜕变都让猎杀之力更强。

## 十、设计参考要点（供绘图/建模）

1. **犬科感**：四足、流线躯干、狭长头部，但保持机械感
2. **攻击性**：前倾姿态、利齿突出、足尖爪钩
3. **机动感**：肩部推进器、尾端平衡杆、肌肉线条般的机械结构
4. **差异化**：与地渊（蚯蚓）、俯冲（鸟）、缚网（蜘蛛）在形态上明显区分
