# 沙蝎 (Sand Stalker) — 详细设计文档

> 沙漠/极地环境仿生机械兽 · 恶劣战场之眼

---

| 项目 | 内容 |
|------|------|
| **中文名** | 沙蝎 |
| **英文名** | Sand Stalker |
| **类型** | 沙漠/极地环境仿生机械兽 |
| **定位** | 恶劣环境侦察与区域骚扰单位 |
| **设计灵感** | 蝎子（沙漠、尾刺、潜行） |

---

## 二、外观设计

### 2.1 整体形态

- **主体结构**：仿蝎子形态，八足 + 双钳 + 尾刺，体长约 2～2.5 米，宽约 1～1.5 米
- **比例**：躯干扁平，双钳可收拢，尾刺可弯曲
- **姿态**：沙地/雪地潜行时贴近地面，尾刺可抬起攻击

### 2.2 配色方案

- **主色**：沙黄（#c4a35a～#d4b896）— 沙漠、雪地融入
- **辅色**：墨绿（#2d5016～#4a7c23）— 迷彩、隐蔽
- **点缀**：幽绿（#32cd32）— 尾刺毒液、传感器

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **头部** | 复眼传感器，适应强光与极寒环境 |
| **双钳** | 可夹持、挖掘，也可用于近战 |
| **八足** | 足尖带沙地/雪地适应结构，潜行时几乎无声 |
| **尾刺** | 可释放神经毒素或电磁脉冲，瘫痪目标 |

### 2.4 材质与质感

- 外壳：耐磨复合装甲，适应沙尘与极寒
- 尾刺：可替换毒素/电磁模块
- 足部：沙地/雪地适应垫，减少痕迹

---

## 三、核心能力

### 3.1 恶劣环境侦察

- 可适应沙漠、极地等极端环境
- 沙尘伪装罩可融入环境，难以被探测
- 适合在恶劣战场进行侦察

### 3.2 区域骚扰

- 可潜行至敌阵侧翼，进行骚扰
- 尾刺可瘫痪关键目标，制造混乱
- 适合配合主攻单位形成多线打击

### 3.3 单点瘫痪

- 尾刺神经毒素可瘫痪轻型单位
- 电磁脉冲可瘫痪电子设备
- 适合斩首、破袭关键目标

### 3.4 地形适应

- 可在沙地、雪地、高温、极寒环境中作战
- 沙尘伪装罩可减少暴露
- 是恶劣战场的侦察与骚扰单位

## 五、职责与战术

### 5.1 战场职责

- 恶劣环境侦察 — 在沙漠、极地侦察
- 区域骚扰 — 潜行骚扰敌阵侧翼
- 单点瘫痪 — 尾刺瘫痪关键目标
- 地形适应 — 适应极端环境作战

### 5.2 战术风格

- **潜行优先**：以隐蔽与骚扰为主
- **协同定位**：为利刃、猎犬等提供恶劣战场支援
- **恶劣专家**：专为沙漠与极地设计

### 5.3 适用地形

- 沙漠、极地、雪地、高温/极寒环境
- 不适合：水域（沧龙负责）、高空、密林（缚网更优）

## 六、等级进化体系

| 等级 | 称号 | 进化要点 |
|------|------|----------|
| 0 | 量产型 | 基础恶劣环境适应能力，可执行侦察任务 |
| 1 | 改良型 | 沙尘伪装能力增强，潜行效率提升 |
| 2 | 特装型 | 尾刺初显，可进行单点瘫痪 |
| 3 | 强化型 | 区域骚扰能力成型，地形适应显著提升 |
| 4 | 沙隐型 | 沙尘伪装罩加身，恶劣环境潜行之威尽显 |
| 5 | 极寒型 | 极地适应强化，可在雪地/极寒中作战 |
| 6 | 毒刺型 | 沙蝎之力臻于完美，恶劣战场侦察与骚扰达到巅峰 |
| 7 | 完整体 | 恶劣环境的终极形态，沙尘伪装与神经毒素尾刺全开，成为沙漠与极地战场的潜行专家 |

## 九、简介文案（用于系统）

**简短描述**：沙漠/极地环境仿生机械兽，擅长恶劣环境侦察与区域骚扰。

**完整介绍**：沙蝎专为沙漠与极地设计，仿蝎子形态，适应高温与极寒，擅长在沙地、雪地中潜行，尾刺可释放毒素或电磁脉冲，兼具侦察与骚扰能力。负责恶劣环境侦察、区域骚扰、单点瘫痪、地形适应。能在极端环境中隐蔽行动，是恶劣战场的侦察与骚扰单位。它在历练中不断觉醒，解锁更强形态、沙尘伪装罩与神经毒素尾刺，每一次蜕变都让沙蝎之力更强。

## 十、设计参考要点（供绘图/建模）

1. **蝎子感**：八足、双钳、尾刺
2. **恶劣感**：沙黄迷彩、耐磨装甲、适应结构
3. **潜行感**：扁平躯干、沙尘伪装、低调配色
4. **差异化**：与缚网（蜘蛛）、猎犬（犬）、地渊（蚯蚓）在形态上明显区分
