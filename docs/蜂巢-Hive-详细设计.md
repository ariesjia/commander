# 蜂巢 (Hive) — 详细设计文档

> 无人机航母载具机甲 · 蜂群之主

---

## 一、基础设定

| 项目 | 内容 |
|------|------|
| **中文名** | 蜂巢 |
| **英文名** | Hive |
| **类型** | 无人机航母载具机甲 |
| **定位** | 无人机部署与多线作战单位 |
| **设计灵感** | 蜂巢、航母（蜂群、指挥、多线） |

---

## 二、外观设计

### 2.1 整体形态

- **主体结构**：中型载具形态，车身呈六边形/蜂巢状，长约 4～6 米，宽约 3～4 米
- **比例**：主体为指挥中枢，背部/侧面有蜂群发射舱，整体呈「蜂巢」姿态
- **装甲**：轻量化，以指挥与部署为主，防护适中

### 2.2 配色方案

- **主色**：黑金（#1a1a1a～#2d2d2d + #d4af37）— 蜂巢、主宰感
- **辅色**：琥珀黄（#ffbf00～#ffd700）— 蜂群、能量
- **点缀**：深褐（#3d2914）— 蜂巢纹路、结构

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **主体** | 六边形/蜂巢状结构，中枢指挥模块居中 |
| **蜂群发射舱** | 背部/侧面可展开，释放小型无人机 |
| **传感器阵列** | 多角度布置，接收蜂群回传数据 |
| **足部/底盘** | 可驻守或缓慢机动，本体不直接交战 |

### 2.4 材质与质感

- 外壳：复合装甲，表面有蜂巢纹路
- 发射舱：可展开舱门，快速释放无人机
- 无人机：小型、灵活，呈蜂群状

---

## 三、核心能力

### 3.1 无人机部署

- 可部署小型无人机群，扩展战场视野与火力
- 蜂群发射舱支持快速释放与回收
- 实现一人多线的战术覆盖

### 3.2 蜂群侦察

- 无人机可进行多方向侦察
- 回传数据至中枢，形成战场态势
- 适合大范围情报获取

### 3.3 协同火力

- 无人机可携带轻型武器，进行火力补充
- 蜂群可集火单一目标，或分散骚扰
- 适合配合主攻单位形成多线打击

### 3.4 多线牵制

- 蜂群可同时牵制多个方向
- 本体作为指挥中枢，协调蜂群行动
- 适合复杂战场、多线作战

## 五、职责与战术

### 5.1 战场职责

- 无人机部署 — 释放蜂群，扩展战场
- 蜂群侦察 — 多方向侦察，回传情报
- 协同火力 — 蜂群火力补充
- 多线牵制 — 同时牵制多个方向

### 5.2 战术风格

- **召唤型**：以蜂群为主，本体不直接交战
- **协同定位**：为利刃、雷啸等提供视野与火力补充
- **一人多线**：通过蜂群实现多线战术覆盖

### 5.3 适用地形

- 开阔战场、复杂地形
- 需星盾、镇岳等掩护本体
- 不适合：狭窄空间（蜂群难以展开）、强电磁干扰

## 九、简介文案（用于系统）

**简短描述**：无人机航母载具，可部署蜂群协同作战。

**完整介绍**：蜂巢是召唤型机甲，通过无人机群扩展战场视野与火力，实现一人多线的战术覆盖。负责无人机部署、蜂群侦察、协同火力、多线牵制。本体作为指挥中枢，通过蜂群实现多线作战。它在历练中不断觉醒，解锁更强形态、蜂群发射舱与中枢指挥模块，每一次蜕变都让蜂群之力更强。

## 十、设计参考要点（供绘图/建模）

1. **蜂巢感**：六边形结构、蜂群纹路、琥珀黄
2. **航母感**：发射舱、指挥中枢、多线部署
3. **主宰感**：黑金配色、中枢模块、蜂群环绕
4. **差异化**：与缚网（蜘蛛）、雷鸦（电子战）在形态上明显区分
