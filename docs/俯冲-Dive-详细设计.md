## 二、外观设计

### 2.1 整体形态

- **主体结构**：仿游隼形态，流线型机身，翼展约 3～4 米，体长约 2 米
- **比例**：头部尖锐，躯干紧凑，双翼可折叠，尾翼可调节
- **姿态**：巡航时翼展平铺，俯冲时收翼呈梭形，减少阻力

### 2.2 配色方案

- **主色**：银白（#e8e8e8～#f5f5f5）— 高空隐蔽、云层融入
- **辅色**：深蓝（#1e3a5f～#2c5282）— 翼缘、腹部、传感器
- **点缀**：亮蓝（#3182ce）— 眼部、推进器喷口

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **头部** | 尖锐流线型，喙部为动能穿甲弹发射口，眼部为高精度瞄准系统 |
| **躯干** | 紧凑装甲，腹部有俯冲加速翼的折叠收纳舱 |
| **双翼** | 可折叠翼面，翼尖可展开辅助推进器，俯冲时收拢减阻 |
| **尾翼** | 可调角度，用于俯冲姿态控制与急停转向 |

### 2.4 材质与质感

- 外壳：光滑复合装甲，减少空气阻力
- 翼面：轻质高强材料，支持高速俯冲
- 喷口：耐高温合金，俯冲时喷口呈亮蓝辉光

---

## 三、核心能力

### 3.1 高速俯冲

- 从高空以极陡角度俯冲，速度可达数倍音速
- 俯冲加速翼展开时提供额外推力
- 俯冲轨迹可微调，锁定移动目标

### 3.2 动能冲击

- 俯冲末端以机身或专用冲角撞击目标，动能转化为破坏力
- 适合打击固定目标、轻型载具、防空阵地
- 一击即离，不滞留低空

### 3.3 精确轰炸

- 动能穿甲弹可在俯冲过程中发射，配合轨迹实现精准命中
- 支持单点打击与小型集群打击
- 适合斩首、破袭高价值目标

### 3.4 快速撤离

- 俯冲攻击后立即拉起，利用惯性爬升
- 翼尖推进器辅助快速脱离低空威胁区
- 不与敌防空火力纠缠

## 四、专属装备

| 装备名 | 功能 |
|--------|------|
| **俯冲加速翼** | 腹部可展开，俯冲时提供额外推力，提升末端速度 |
| **动能穿甲弹** | 喙部发射，配合俯冲轨迹精准命中，可穿透轻型装甲 |
| **高精度瞄准系统** | 头部眼部，锁定地面目标，计算俯冲轨迹 |
| **翼尖推进器** | 双翼末端，辅助爬升与急停转向 |

## 五、职责与战术

### 5.1 战场职责

- 俯冲打击 — 从高空俯冲攻击地面目标
- 对地突袭 — 快速切入敌阵后方或薄弱环节
- 精确轰炸 — 定点清除高价值目标
- 快速撤离 — 攻击后立即脱离，避免被防空火力锁定

### 5.2 战术风格

- **游隼型**：高空盘旋→锁定目标→俯冲→攻击→拉起撤离
- **一击即离**：不恋战，单次攻击后立即脱离
- **协同定位**：为空境、雷啸等提供对地火力补充

### 5.3 适用地形

- 开阔战场、平原、沙漠
- 城市上空（需注意建筑物遮挡）
- 不适合：密林、地下、深海

## 七、人物特质（可选）

- **性格**：冷静、精准、追求一击必中
- **战斗风格**：高空俯瞰，锁定后俯冲而下，不拖泥带水
- **与驾驶员**：需要高度默契，俯冲时机与轨迹需精确配合

## 九、简介文案（用于系统）

**简短描述**：仿生猛禽突击型机甲，擅长高速俯冲攻击与精准打击。

**完整介绍**：俯冲是空中对地突击单位，以高速俯冲与精准打击压制地面目标，一击即离。负责俯冲打击、对地突袭、精确轰炸、快速撤离。它在历练中不断觉醒，解锁更强形态、俯冲加速翼与动能穿甲弹，每一次蜕变都让俯冲之力更强。

## 十、设计参考要点（供绘图/建模）

1. **猛禽感**：流线机身、尖锐喙部、收翼俯冲姿态
2. **速度感**：梭形轮廓、翼面线条、推进器喷口
3. **精准感**：瞄准系统、穿甲弹发射口、可调尾翼
4. **差异化**：与空境（大型载具）、猎犬（四足）、缚网（蜘蛛）在形态上明显区分
