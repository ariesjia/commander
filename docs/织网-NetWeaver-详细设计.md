# 织网 (Net Weaver) — 详细设计文档

> 信息战/电子战型人形机甲 · 无形之网

---

## 一、基础设定

| 项目 | 内容 |
|------|------|
| **中文名** | 织网 |
| **英文名** | Net Weaver |
| **类型** | 信息战/电子战型人形机甲 |
| **定位** | 电子入侵与战场网络控制单位 |
| **设计灵感** | 黑客、蜘蛛（入侵、织网、无形） |

---

## 二、外观设计

### 2.1 整体形态

- **主体结构**：标准人形机甲，体型纤细，身高约 2.8～3.2 米，背部有天线阵列
- **比例**：四肢修长，躯干紧凑，整体呈「网络节点」姿态
- **装甲**：轻量化，以电子战设备为主

### 2.2 配色方案

- **主色**：深蓝（#1e3a5f～#2c5282）— 科技、网络感
- **辅色**：荧光青（#00ced1～#40e0d0）— 数据流、能量
- **点缀**：银白（#e8e8e8）— 天线、传感器

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **头部** | 传感器阵列突出，多频段天线 |
| **躯干** | 神经入侵模块，可释放电磁脉冲 |
| **双臂** | 可展开入侵接口，连接敌方系统 |
| **背部** | 天线阵列，扩大入侵与干扰范围 |

### 2.4 材质与质感

- 外壳：轻质复合装甲，表面有数据流纹路
- 天线：可伸缩，释放时呈荧光青辉光
- 入侵模块：科技感强，连接时呈数据流

---

## 三、核心能力

### 3.1 电子入侵

- 可入侵敌方通讯、雷达、火控系统
- 神经入侵模块支持远程与近距离入侵
- 在无形层面压制对手

### 3.2 系统瘫痪

- 可瘫痪敌方机甲、载具、防御设施
- 电磁脉冲手雷可范围瘫痪
- 适合削弱敌方战力

### 3.3 通讯干扰

- 可干扰敌方通讯，切断指挥链
- 使敌人无法协同作战
- 适合配合利刃、猎犬实施突袭

### 3.4 数据窃取

- 可窃取敌方情报、地图、部署信息
- 为友军提供战场态势
- 是数字化战场的核心节点

| 装备名 | 功能 |
|--------|------|
| **神经入侵模块** | 躯干核心，可入侵敌方系统 |
| **电磁脉冲手雷** | 可投掷，范围瘫痪电子设备 |
| **多频段天线** | 背部与头部，扩大入侵与干扰范围 |
| **数据流接口** | 双臂，可连接敌方系统进行入侵 |

---

## 五、职责与战术

### 5.1 战场职责

- 电子入侵 — 入侵敌方系统
- 系统瘫痪 — 瘫痪敌方机甲与设施
- 通讯干扰 — 切断敌方通讯
- 数据窃取 — 窃取敌方情报

### 5.2 战术风格

- **无形优先**：不依赖火力，以入侵与干扰为主
- **协同定位**：为利刃、猎犬、雷啸等创造优势
- **数字化核心**：是数字化战场的核心节点

### 5.3 适用地形

- 数字化战场、城市、有通讯网络的区域
- 不适合：无电子设备的原始战场、强电磁屏蔽

## 六、等级进化体系

| 等级 | 称号 | 进化要点 |
|------|------|----------|
| 0 | 量产型 | 基础电子入侵能力，可执行通讯干扰任务 |
| 1 | 改良型 | 入侵范围与效率增强 |
| 2 | 特装型 | 系统瘫痪能力初显 |
| 3 | 强化型 | 数据窃取能力成型，战场网络控制初显 |
| 4 | 入侵型 | 神经入侵模块加身，入侵与瘫痪能力大幅增强 |
| 5 | 织网型 | 战场网络控制觉醒，无形压制之威尽显 |
| 6 | 主宰型 | 织网之力臻于完美，数字化战场核心达到巅峰 |
| 7 | 完整体 | 电子战的终极形态，神经入侵模块与电磁脉冲全开，成为数字化战场的无形主宰 |

## 九、简介文案（用于系统）

**简短描述**：信息战/电子战型人形机甲，专攻电子入侵与战场网络控制。

**完整介绍**：织网是信息战专家，专攻电子入侵与战场网络控制，可瘫痪敌方系统、窃取数据、干扰通讯，在无形层面压制对手。负责电子入侵、系统瘫痪、通讯干扰、数据窃取。不依赖火力，是数字化战场的核心节点。它在历练中不断觉醒，解锁更强形态、神经入侵模块与电磁脉冲手雷，每一次蜕变都让织网之力更强。

## 十、设计参考要点（供绘图/建模）

1. **科技感**：天线阵列、数据流纹路、荧光青
2. **无形感**：轻量化、传感器突出、入侵接口
3. **网络感**：深蓝配色、多频段天线、神经模块
4. **差异化**：与缚网（蜘蛛渗透）、磁暴（磁场）、雷啸（雷电）在形态上明显区分
