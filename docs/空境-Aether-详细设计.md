# 空境 (Aether) — 详细设计文档

> 空中全域载具机甲 · 天穹之翼

---

| 项目 | 内容 |
|------|------|
| **中文名** | 空境 |
| **英文名** | Aether |

| **定位** | 空中运输与全域支援单位 |
| **设计灵感** | 运输机、飞艇（运输、跃迁、支援） |

---

## 二、外观设计

### 2.1 整体形态

- **主体结构**：大型飞行载具形态，流线型机身，翼展约 15～20 米，机长约 12～15 米
- **比例**：宽体机身，可容纳多名乘员或机甲，双翼可折叠
- **姿态**：巡航时翼展平铺，跃迁时收翼呈梭形

### 2.2 配色方案

- **主色**：银白（#e8e8e8～#f5f5f5）— 高空、云层融入
- **辅色**：天蓝（#4a90d9～#6eb5e8）— 引擎、跃迁装置
- **点缀**：星辰金（#d4af37）— 跃迁核心、纹饰

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **机首** | 流线型，驾驶舱与传感器阵列 |
| **机身** | 宽体货舱，可搭载机甲或物资 |
| **双翼** | 可折叠翼面，翼下挂载跃迁引擎 |
| **尾翼** | 可调角度，用于姿态控制 |

### 2.4 材质与质感

- 外壳：轻质复合装甲，表面光滑减阻
- 引擎：跃迁引擎呈天蓝辉光
- 货舱：可展开舱门，快速部署

---

## 三、核心能力

### 3.1 高空穿梭

- 可在高空高速飞行，跨越地形障碍
- 跃迁引擎支持短距跃迁，快速抵达目标区域
- 是团队主要的空中机动单位

### 3.2 快速部署

- 可搭载机甲与物资，快速投送至战场
- 舱门可展开，支持空投与机降
- 适合全员机动转移、紧急支援

### 3.3 全域支援

- 可跨越陆地、海洋、山地，实现全域机动
- 为友军提供运输、补给、撤离支援
- 是战场的「空中枢纽」

### 3.4 高空侦查

- 配备高空传感器，可进行远距离侦察
- 为友军提供战场态势
- 适合大范围情报获取

## 四、专属装备

| 装备名 | 功能 |
|--------|------|
| **天境引擎** | 主推进系统，支持高空高速飞行 |
| **跃迁引擎** | 翼下挂载，支持短距跃迁 |
| **宽体货舱** | 可搭载机甲、物资、人员 |
| **高空传感器** | 机首与机身，远距离侦察 |

---

## 五、职责与战术

### 5.1 战场职责

- 空中运输 — 搭载机甲与物资，快速投送
- 全域支援 — 跨越地形，提供支援
- 高空侦查 — 远距离侦察，提供态势

### 5.2 战术风格

- **支援优先**：以运输与支援为主，不直接交战
- **协同定位**：为俯冲、风驰等提供投送与情报
- **机动核心**：是团队空中机动的枢纽

### 5.3 适用地形

- 全域适用，以空中为主
- 不适合：地下、深海、狭窄空间

## 六、等级进化体系

| 等级 | 称号 | 进化要点 |
|------|------|----------|
| 0 | 量产型 | 基础高空穿梭能力，可执行空中运输任务 |
| 1 | 改良型 | 快速部署能力增强，浮空之能初成 |
| 2 | 特装型 | 全域载具之能初显，可跨越地形机动转移 |
| 3 | 强化型 | 跃迁之力凝实，全域支援与高空侦查能力成型 |
| 4 | 跃迁型 | 跃迁引擎加身，高空穿梭与快速部署能力大幅增强 |
| 5 | 超跃型 | 跃迁核心觉醒，团队运输之威尽显 |
| 6 | 破穹型 | 穿梭之力臻于完美，跨越地形实现全员机动转移达到巅峰 |
| 7 | 完整体 | 空中全域的终极形态，跃迁引擎与天境核心全开，成为团队空中机动的枢纽 |

## 七、人物特质（可选）

- **性格**：沉稳、可靠、以支援为使命
- **战斗风格**：不争锋芒，默默承担运输与支援
- **与驾驶员**：需要全局视野，投送时机需配合战场态势

## 九、简介文案（用于系统）

**简短描述**：空中全域载具机甲，擅长高空穿梭与快速部署。

**完整介绍**：空境，空中全域载具机甲，擅长高空穿梭与快速部署，可跨越地形实现全员机动转移。负责空中运输、全域支援、高空侦查。它在历练中不断觉醒，解锁更强形态、天境引擎与跃迁之力，每一次蜕变都让穿梭之力更强。

## 十、设计参考要点（供绘图/建模）

1. **载具感**：宽体机身、货舱、可折叠翼
2. **空中感**：流线型、银白配色、跃迁引擎
3. **支援感**：舱门、投送装置、传感器
4. **差异化**：与俯冲（小型猛禽）、沧龙（两栖）在形态上明显区分
