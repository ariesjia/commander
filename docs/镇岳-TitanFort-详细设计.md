# 镇岳 (Titan Fort) — 详细设计文档

> 陆地堡垒载具机甲 · 移动要塞

---

## 一、基础设定

| 项目 | 内容 |
|------|------|
| **中文名** | 镇岳 |
| **英文名** | Titan Fort |
| **类型** | 陆地堡垒载具机甲 |
| **定位** | 阵地驻守与重型防御单位 |
| **设计灵感** | 坦克、堡垒（防御、驻守、火力） |

---

## 二、外观设计

### 2.1 整体形态

- **主体结构**：重型陆地载具形态，车身宽厚，长约 6～8 米，宽约 3～4 米，高约 2.5～3 米
- **比例**：低矮宽体，底盘稳固，炮塔居中，整体呈「堡垒」姿态
- **装甲**：全身超厚装甲，正面尤为突出

### 2.2 配色方案

- **主色**：深灰（#3d3d3d～#5c5c5c）— 沉稳、防御感
- **辅色**：橄榄绿（#4a5d23～#6b8e23）— 陆地、隐蔽
- **点缀**：金属银（#808080）— 装甲接缝、炮管

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **车首** | 楔形正面装甲，可承受正面冲击 |
| **车身** | 宽体底盘，可搭载乘员与火力模块 |
| **炮塔** | 可旋转炮塔，主炮 + 副武器 |
| **履带** | 超宽履带，适应各种陆地地形 |

### 2.4 材质与质感

- 外壳：超重型复合装甲，表面略粗糙
- 炮塔：可展开防御屏障，增强驻守能力
- 履带：宽幅设计，泥地、雪地均可通行

---

## 三、核心能力

### 3.1 阵地驻守

- 可稳定驻守关键节点，难以被击退
- 车身坚固如堡垒，可承受猛烈攻击
- 是陆地上最可靠的移动防御堡垒

### 3.2 防线构筑

- 可为团队构筑安全防线
- 配合星盾形成双重防御
- 适合防守战、撤离掩护

### 3.3 重型防御

- 超厚装甲可抵御高强度攻击
- 可为队友提供坚固掩护
- 适合吸引火力、保护关键单位

### 3.4 火力压制

- 主炮与副武器可进行阵地压制
- 火力覆盖可压制敌方推进
- 适合固守阵地、反击推进

### 3.5 战场固守

- 可在任何陆地战场驻守防线
- 稳定性极强，难以被撼动
- 是团队陆上防御的核心

## 五、职责与战术

### 5.1 战场职责

- 阵地驻守 — 稳定驻守关键节点
- 防线构筑 — 为团队构筑安全防线
- 重型防御 — 抵御攻击，掩护队友
- 团队掩护 — 吸引火力，保护友军
- 战场固守 — 在任何陆地战场驻守
- 火力压制 — 阵地压制与反击

### 5.2 战术风格

- **驻守优先**：以固守阵地为主，不主动推进
- **协同定位**：为玄甲、雷啸等创造输出空间
- **移动要塞**：是陆地上绝对的移动防御堡垒

### 5.3 适用地形

- 陆地全地形：平原、山地、沙漠、雪地
- 不适合：水域（沧龙负责）、高空、地下

## 六、等级进化体系

| 等级 | 称号 | 进化要点 |
|------|------|----------|
| 0 | 量产型 | 基础阵地驻守能力，可承担掩护队友推进任务 |
| 1 | 改良型 | 强力防御能力增强，可抵御一定强度攻击 |
| 2 | 特装型 | 陆地堡垒之能初显，可为团队构筑安全防线 |
| 3 | 强化型 | 防御之力凝实，阵地驻守与掩护推进能力成型 |
| 4 | 壁垒型 | 全域防御屏障加身，抵御高强度攻击能力大幅增强 |
| 5 | 重装型 | 超重型防御装甲觉醒，堡垒驻守之威尽显 |
| 6 | 要塞型 | 防御之力臻于完美，陆地上最可靠的移动防御堡垒 |
| 7 | 完整体 | 陆地堡垒的终极形态，超重型装甲与壁垒屏障全开，成为陆地上最可靠的移动要塞 |

## 七、人物特质（可选）

- **性格**：沉稳、坚韧、以固守为使命
- **战斗风格**：不争锋芒，默默驻守防线
- **与驾驶员**：需要耐心，驻守任务往往长时间不变

## 九、简介文案（用于系统）

**简短描述**：陆地堡垒载具机甲，擅长阵地驻守与强力防御。

**完整介绍**：镇岳，陆地堡垒载具机甲，擅长阵地驻守与强力防御，车身坚固无比，能抵御高强度攻击，为团队构筑安全防线；可稳定驻守战场、掩护队友推进，是陆地上最可靠的移动防御堡垒。它在历练中不断觉醒，解锁更强形态、超重型防御装甲与壁垒之力，每一次蜕变都让防御之力更强。

## 十、设计参考要点（供绘图/建模）

1. **堡垒感**：宽体、低矮、超厚装甲
2. **陆地感**：履带、橄榄绿、稳固底盘
3. **火力感**：炮塔、主炮、副武器
4. **差异化**：与星盾（人形防御）、沧龙（两栖）、空境（空中）在形态上明显区分
