# 铁龙 (Iron Dragon) — 详细设计文档

> 多节列车载具机甲 · 陆上轨道之龙

---

## 一、基础设定

| 项目 | 内容 |
|------|------|
| **中文名** | 铁龙 |
| **英文名** | Iron Dragon |
| **类型** | 多节列车载具机甲 |
| **定位** | 轨道运输与编队火力单位 |
| **设计灵感** | 装甲列车、多节货运列车（运输、编队、火力） |

---

## 二、外观设计

### 2.1 整体形态

- **主体结构**：多节列车形态，车头长约 8～10 米，单节车厢约 6～8 米，可组合为 3～6 节编队
- **比例**：长条形车身，车头略高呈楔形，车厢呈矩形箱体，整体呈「铁龙」姿态
- **装甲**：全身重型装甲，车头与车厢连接处可拆解

### 2.2 配色方案

- **主色**：墨绿（#2d4a2d～#3d5c3d）— 沉稳、陆地感
- **辅色**：铜金（#b87333～#cd7f32）— 连接件、炮管、纹饰
- **点缀**：金属银（#808080）— 轨道、轮毂

### 2.3 结构细节

| 部位 | 设计要点 |
|------|----------|
| **车头** | 楔形正面，驾驶舱与传感器，多联装轨道炮 |
| **车厢** | 可拆分模块，可搭载机甲或火力单元 |
| **连接处** | 电磁/机械耦合，支持快速拆解与重组 |
| **底盘** | 轨道轮组，适应铁路与专用轨道 |

### 2.4 材质与质感

- 外壳：重型复合装甲，表面略粗糙
- 车厢：模块化设计，可挂载不同功能单元
- 轨道：宽幅轮组，适应各种轨距

---

## 三、核心能力

### 3.1 多节运输

- 可搭载多台机甲或大量物资，沿铁路快速投送
- 车厢可拆分，支持多点投送
- 是陆上长距离机动的主要运输单位

### 3.2 编队火力

- 多节车厢可装备火力模块，形成编队火力压制
- 车头主炮与车厢副炮协同覆盖
- 适合铁路沿线压制、阵地封锁

### 3.3 阵地压制

- 多联装轨道炮可进行持续火力压制
- 编队协调系统统一射击节奏
- 适合固守铁路节点、封锁通道

### 3.4 铁路封锁

- 可驻守铁路沿线，阻断敌方机动
- 车厢可拆解布设封锁点
- 是陆上轨道战场的核心单位

---

## 四、专属装备

| 装备名 | 功能 |
|--------|------|
| **可拆分车厢** | 模块化设计，支持快速拆解与重组，每节可独立作战 |
| **多联装轨道炮** | 车头主炮，持续火力压制 |
| **编队协调系统** | 统一多节车厢的火力与机动节奏 |

---

## 五、职责与战术

### 5.1 战场职责

- 多节运输 — 搭载机甲与物资，沿铁路快速投送
- 编队火力 — 多节协同，形成火力压制
- 阵地压制 — 铁路沿线压制，封锁通道
- 铁路封锁 — 驻守铁路节点，阻断敌方机动

### 5.2 战术风格

- **编队优先**：以编队协同为主，不单独作战
- **协同定位**：为地渊、方舟等提供轨道运输与火力支援
- **轨道核心**：是陆上轨道战场的重型单位

### 5.3 适用地形

- 铁路沿线、专用轨道、平原轨道
- 不适合：无轨道区域、山地、水域、高空

## 六、等级进化体系

| 等级 | 称号 | 进化要点 |
|------|------|----------|
| 0 | 量产型 | 基础轨道运输能力，可执行多节运输任务 |
| 1 | 改良型 | 编队协调能力增强，车厢衔接更加稳固 |
| 2 | 特装型 | 多联装轨道炮初显，可承担阵地压制重任 |
| 3 | 强化型 | 可拆分车厢能力成型，编队火力与机动显著提升 |
| 4 | 多节型 | 多节编队加身，大规模投送与铁路封锁能力大幅增强 |
| 5 | 编队型 | 编队协调系统觉醒，陆上重型轨道之威尽显 |
| 6 | 铁龙型 | 轨道之力臻于完美，多节运输与阵地压制达到巅峰 |
| 7 | 完整体 | 多节列车的终极形态，可拆分车厢与编队协调之力全开，成为陆上重型轨道单位 |

## 七、人物特质（可选）

- **性格**：沉稳、威严、以编队与秩序为使命
- **战斗风格**：不争锋芒，以编队协同与阵地压制为主
- **与驾驶员**：需要全局视野，编队调度需配合战场态势

## 八、与其他机甲的协同

| 机甲 | 协同方式 |
|------|----------|
| 地渊 | 地渊开辟地下隧道连接铁路，铁龙沿轨道快速投送 |
| 方舟 | 铁龙运输方舟至前线，方舟补给友军 |
| 镇岳 | 铁龙投送镇岳至阵地，镇岳驻守防线 |
| 空境 | 空境负责空中投送，铁龙负责陆上轨道投送，形成立体运输 |

## 九、简介文案（用于系统）

**简短描述**：多节列车载具机甲，车厢可拆分重组，兼具运输与火力。

**完整介绍**：铁龙，多节列车载具机甲，车厢可拆分重组，兼具运输与火力。负责大规模投送、编队火力、铁路封锁，是陆上重型轨道单位。它在历练中不断觉醒，解锁更强形态、可拆分车厢与编队协调之力，每一次蜕变都让轨道之力更强。

## 十、设计参考要点（供绘图/建模）

1. **列车感**：长条形车身、多节车厢、轨道轮组
2. **重型感**：墨绿装甲、铜金点缀、多联装炮
3. **模块感**：可拆分连接处、车厢接口、编队协调天线
4. **差异化**：与镇岳（履带堡垒）、空境（空中）、沧龙（两栖）在形态上明显区分
